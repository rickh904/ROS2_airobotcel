# Een andere uFactory-type robot configureren.

Je kunt deze uFactory repository ook gebruiken voor andere uFactory-type robots, zoals de uFactory xArm 6 of uFactory xArm 7. De configuratie van deze robots is vergelijkbaar met de configuratie van de uFactory Lite robot, maar er zijn enkele verschillen in de netwerkconfiguratie en de configuratie van de robot zelf. 
Je kunt deze template repository gebruiken als basis voor het configureren van een andere uFactory-type robot.

:::{tip}
Zorg ervoor dat je nieuwe configuratie unieke package-namen heeft, zodat er geen conflicten ontstaan met de configuratie van de uFactory Lite6 robot.
:::
