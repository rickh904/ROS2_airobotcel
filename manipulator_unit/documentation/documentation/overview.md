# Inleiding

Welkom bij de template voor het maken van `uFactory Lite6 ROS2-Jazzy` projecten. 

Deze template biedt de mogelijkheid om op een relatief eenvoudige wijze een applicatie voor een uFactory Lite6 robot te bouwen waarbij een virtuele wereld, gemodelleerd in URDF, gepresenteerd kan worden in de RVIZ monitor of gesimuleerd met Gazebo.

Deze template bestaat uit een aantal ROS2 packages met een prefix `my_uf` om onderscheid te maken met andere uFactory Lite6 ROS2 packages. 

In deze documentatie wordt beschreven hoe de template geinstalleerd kan worden, er met een fysieke robot gewerkt kan worden en hoe de applicatie kan worden gesimuleerd. Tevens wordt beschreven hoe de template toegepast kan worden voor een `custom-made` project.

De template is toegespitst op een `uFactory Lite6` robot, maar kan zonder al te veel inspanningen voor een ander type robot van uFactory worden gemigreerd.
